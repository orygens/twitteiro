h1. Twitteiro

h2. Resolver as dependências do aplicativo
* cd no diretório do projeto e execute $ bundle install - Isso instalará todas as dependencias do projeto.
* Execute $ rake db:create e rake db:migrate pra migrar o BD.

h2. Iniciar o servidor
* cd no diretório do projeto e execute $ thin start -d, em seguida acesse "http://localhost:3000/":http://localhost:3000/

h2. Tarefas do app

* rake about - Mostra informações do ambiente do projeto.
* rake stats - Mostra estatísticas do código do app.
* rake routes - Mostra todas as urls do app.
* rake spec - Executa a suíte de testes com RSpec.
* rake test - Executa a suíte de testes com Test::Unit e Shoulda.
* rails c - Abre um console interativo do Ruby com o aplicativo carregado.
